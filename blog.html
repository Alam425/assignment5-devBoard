Question-1: What are the different ways to select an element in the DOM?

=> There are some methods to do it. Ex.- document.getElementByTagName(), document.getElementById(), document.getElementByClassName(), document.querySelector(), document.querySelectorAll() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

=> innerHTML gives the whole of the html. innerText gives the text inside an element only. textContent gives the text but also the hidden ones.

Question-3: What is event delegation in the DOM?

=> If any event(click, hover, mouseenter etc.) occurs in any childNode, it need to be captured. Then, it needs to be bubbled up to the parent to do something to it. This is DOM Event Delegation.

Question-4: What is event bubbling in the DOM?

=> After capturing an event, it is sent to the parent. This is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

=> You can create any element by document.createElement() method, add any element by document.appendChild() method, remove any element by document.removeChild() method.
